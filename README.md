# Microsserviços com Spring Boot e Spring Cloud
### Curso realizado durante o curso da Udemy do professor Nélio Alves.

O projeto é um gerado básico de folha de pagamento para um Departamento de Recursos Humanos de uma empresa. E tudo foi implementado utilizando a estrutura de microsserviços.

## Back-end

- Spring Boot
- Spring Cloud
  - OAuth2
  - FeignClient
  - Eureka 
  - Ribbon
  - API Gateway Zuul
  - Hystrix
  - Config Server
- JWT
- Docker
